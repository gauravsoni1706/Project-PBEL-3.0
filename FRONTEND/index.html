📄 CSV File Requirements

Please upload a CSV file having the same format used to train the Machine Learning model.

  • ✅ Upload only .csv files.
  • ✅ First row must contain column names.
  • ✅ Required columns are:
Time, V1, V2, V3, V4, V5, V6, V7, V8, V9, V10, V11, V12, V13, V14, V15, V16, V17, V18, V19, V20, V21, V22, V23, V24, V25, V26, V27, V28, Amount
Important:
  • Do not rename any column.
  • Do not remove any column.
  • Do not include the Class column.
  • Only numeric values are accepted.
  • Each row should contain one credit card transaction.
